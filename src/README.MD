# Sangini Web (Landing Page)

The official gateway for the Sangini Trading Platform. This responsive web application is designed to introduce users to virtual trading and financial literacy.

## 🌐 Overview
A minimalist, modern landing page built with React.js. It serves as the primary entry point for user onboarding, highlighting the platform's key features: real-time trading, educational quizzes, and portfolio analytics.

## ⚡ Features
* **User Onboarding:** Seamlessly integrated authentication flow (Login/Signup) that connects to the Sangini Backend.
* **Responsive Architecture:** Fully optimized for mobile and desktop viewing, ensuring a professional experience on any device.
* **Feature Showcasing:** High-level overview of the Trading Terminal and the Financial Literacy Quiz modules to drive user engagement.
* **Optimized Performance:** Fast load times and smooth transitions to create a trustworthy financial atmosphere.

## 🛠 Tech Stack
* **Framework:** React.js
* **Routing:** React Router DOM (v6+)
* **Styling:** Custom CSS3 with focus on typography and high-contrast accessibility.
* **Deployment:** Vercel (CI/CD integration)
